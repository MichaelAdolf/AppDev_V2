# StockMind

StockMind ist eine modulare Aktienanalyse-Plattform.

Ziele:

- tägliche Marktanalyse
- Watchlist-Verwaltung
- technische Analyse
- Fundamentalanalyse
- Backtesting
- Home Assistant Integration
- Node-RED Integration
- Jarvis Integration
- Streamlit Dashboard

Architektur:

- Clean Architecture
- SOLID
- Domain Driven Design
